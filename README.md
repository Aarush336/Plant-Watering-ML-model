# Plant-Watering-ML-model

## Description:
   This project uses machine learning to predict whether a plant needs water based on enviromental sensor data including temperature, humidity, and soil moisture. A Decision Tree classifier processes the sensor readings and determines the optimal watering decision for the plant's health.

## Problem Statement:
   Plants are often overwatered or underwatered due to inconsistent manual monitoring. This model automates that decision by analysing real-time enviromental conditions.

## How it works?
1. Enviromental data is collected- temperature, humidity, soil_moisture.
2. Data is fed into a trained Decision Tree classifier.
3. The model predicts whether the plant is healthy or needs intervention.
4. The result triggers the watering system accordingly.

## Technologies used:
1. Python.
2. Pandas.
3. Numpy.
4. Scikit-learn(Decision Tree classifier).
